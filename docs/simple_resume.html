Notable Open Source Projects

  • Dear PyGui - https://github.com/hoffstadt/DearPyGui

Dear PyGui is a Python GUI framework written in C++ and built with Dear ImGui. The library simulates a traditional retained mode GUI as opposed to Dear ImGui's immediate mode paradigm. The library has continued to grow in popularity approaching 5500 stars, 30+ sponsors, and 500,000 downloads. We currently utilize Directx11, Metal, and OpenGL as the backend graphics APIs.
